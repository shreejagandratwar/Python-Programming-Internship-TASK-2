# 💰 Expense Tracker (Python Project)

A simple command-line based Expense Tracker application built using Python.  
It helps users record, view, and calculate their daily expenses using a CSV file.

## 🚀 Features

- Add new expense with date, description, and amount  
- View all saved expenses  
- Calculate total expense  
- Exit application safely  
- Data stored in CSV file for persistence  

## 🛠️ Technologies Used

- Python  
- CSV Module  
- File Handling  
- DateTime Module  

## 📂 Project Structure
expense-tracker/
│── main.py
│── expenses.csv (auto-created after running program)
│── README.md
